# SQL Injection

This section contains structured labs on SQL Injection from the PortSwigger Web Security Academy.

## Overview

SQL Injection occurs when untrusted input is used to manipulate database queries.

This section focuses on identifying injection points and understanding how query behavior changes under different inputs.

## Covered Techniques

- Authentication bypass
- UNION-based injection
- Error-based SQL Injection
- Blind SQL Injection (Boolean/Time-based)
- Data extraction techniques

## Lab Structure

Each lab includes:
- Injection point identification
- Payload testing
- Exploitation steps
- Observed behavior
- Mitigation notes

## Note

All labs are performed in controlled, intentionally vulnerable environments for educational purposes.
